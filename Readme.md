# Fastify API + TypeScript Boilerplate

This configuration let create Fastify API with TypeScript. 

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Support](#support)
- [Contributing](#contributing)
- [License](#license)

## Installation 

- Run `npm ci`
- Create `.env` or copy from `.env.example`

## Usage

### Dev

Run `npm run start:watch`

### Build 

Run `npm run build`
